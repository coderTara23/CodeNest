# ExpenseFlow 💸

A smart expense tracker built with Python for managing daily expenses efficiently.

## Features
- Add expenses
- View all expenses
- Calculate total expenses
- Save data in CSV
- Expense visualization with charts

## Tech Stack
- Python
- CSV
- Matplotlib
